# Stroop-task
Stroop Task with PsychoPy  with voice detection
https://github.com/jujakoss/Stroop-task

To run :

1 - clone this repository on your local machine

2 - On terminal :

    cd <project_directory>

    pip install -r requirements.txt

    python stroopy.py
